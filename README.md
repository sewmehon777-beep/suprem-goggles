# Bereket Wondimu Personal Website

This is a personal portfolio website built with HTML, CSS, and JavaScript.

## Features
- Responsive design
- Language selector
- Welcome popup (first visit)
- Profile and contact pages

## How to Deploy
1. Push this folder to a GitHub repository.
2. Enable GitHub Pages in the repository settings (set source to `main` branch, `/ (root)` folder).
3. Your site will be live at `https://your-username.github.io/your-repo-name/`.

---

Feel free to customize and use for your own portfolio!
